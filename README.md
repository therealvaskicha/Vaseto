# Vaseto's Personal Webpage

This project is a personal webpage for Vaseto, featuring a minimalistic design with a big "Hi." in the center and a section for projects.

## Instructions for Self-Hosting via Node.js:

1. Ensure you have Node.js installed on your machine.
2. Navigate to the project directory in your terminal.
3. Run `npm install` to install any dependencies listed in `package.json`.
4. Start the server by running `node server.js`.
5. Open your browser and go to `http://localhost:3000` to view your personal webpage.
